## 📂 数据输出

抓取完成后，数据包保存在：

```

├── full_dataset_多芬_*.json       # 原始完整数据

├── market_trends_*.json          # 市场热点
```

**下载方式**：
1. 在界面"数据包下载"区域查看
2. 点击"下载"按钮获取文件
3. 或直接访问 `./data/` 目录

## 🆘 遇到问题？

### 问题 1: 页面打不开

**检查**：
```bash
# 确认服务器运行
lsof -i :8080

# 查看进程
ps aux | grep platform-server
```

**解决**：
```bash
# 重启服务器
npm run platform
```

### 问题 2: 抓取失败

**检查**：
- API Key 是否配置（`node quick_test.js`）
- 网络连接是否正常
- 查看终端日志

### 问题 3: 下载文件为空

**检查**：
```bash
# 查看数据目录
ls -lh data/

# 确认文件内容
cat data/*.json | head -20
```
