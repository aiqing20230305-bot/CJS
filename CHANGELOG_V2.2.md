## ✨ 新功能

### 1. 数据统计面板 ⭐⭐⭐⭐⭐

**痛点**：无法快速了解数据概况，需要逐个下载查看

**解决方案**：
- 在数据包下载区域上方显示统计卡片
- 实时展示关键指标
- 自动更新，无需手动刷新

**核心指标**：
1. **数据包总数** 📦
   - 显示当前所有JSON文件数量
   
2. **总数据量** 💾
   - 自动格式化（KB/MB/GB）
   - 精确到小数点后2位
   
3. **最新抓取** 🕒
   - 相对时间显示（刚刚/X分钟前/X小时前/X天前）
   - 超过7天显示具体日期
   
4. **平均大小** 📊
   - 单个文件的平均大小
   - 帮助评估数据密度

**技术实现**：
```javascript
// 后端 - 统计API
GET /api/stats

Response: {
  "totalFiles": 10,
  "totalSize": 52428800,
  "totalSizeFormatted": "50.00 MB",
  "avgFileSize": "5.00 MB",
  "latestFile": {
    "filename": "market_brand_123.json",
    "time": "2026-04-06T10:30:00.000Z"
  },
  "typeDistribution": {
    "market": 3,
    "videos": 4,
    "full": 3
  },
  "generatedAt": "2026-04-06T11:00:00.000Z"
}

// 前端 - 统计卡片
async function loadStatistics() {
  const stats = await fetch('/api/stats').then(r => r.json());
  // 更新卡片显示
}
```

**UI设计**：
- 网格布局（4列自适应）
- 卡片式设计（Notion风格）
- 悬停动画（轻微上浮）
- 图标 + 数值 + 标签结构
- 响应式布局（小屏自动折叠）

**用户体验**：
- ✅ 一眼了解数据全貌
- ✅ 无需逐个打开文件
- ✅ 自动刷新（下载/抓取后）
- ✅ 相对时间人性化显示

### 新增API端点

| 端点 | 方法 | 说明 | 新增 |
|------|------|------|------|
| `/api/stats` | GET | 返回数据统计信息 | ✅ |

### API响应结构

```json
{
  "totalFiles": 10,           // 文件总数
  "totalSize": 52428800,      // 总大小（字节）
  "totalSizeFormatted": "50.00 MB",  // 格式化大小
  "avgFileSize": "5.00 MB",   // 平均文件大小
  "latestFile": {             // 最新文件
    "filename": "xxx.json",
    "time": "2026-04-06T10:30:00.000Z"
  },
  "typeDistribution": {       // 数据类型分布
    "market": 3,
    "videos": 4,
    "full": 3
  },
  "generatedAt": "2026-04-06T11:00:00.000Z"  // 统计生成时间
}
```
